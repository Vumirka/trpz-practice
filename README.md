## 📖 Про проєкт

> 💜 **NEXIA** - це персональний веб-застосунок для організації робочого простору,
> розроблений у рамках практики з дисципліни ТРПЗ.
>
> Застосунок дозволяє керувати завданнями, нотатками, фінансами, звичками та цілями
> в єдиному зручному інтерфейсі з підтримкою темної теми та збереженням даних у браузері.

---

## 🎯 Постановка задачі

Розробити **односторінковий веб-застосунок** для управління особистим робочим простором з наступними модулями:

- **Дашборд** - загальна статистика та швидкий доступ до всіх розділів
- **Завдання** - менеджер задач з пріоритетами, дедлайнами та підзавданнями
- **Kanban** - візуальна дошка з колонками «До виконання / В процесі / Виконано»
- **Звички** - тижневий трекер корисних звичок зі стріком
- **Цілі** - відстеження прогресу особистих цілей
- **Таймер** - Pomodoro-таймер із записом сесій
- **Замовлення** - менеджер клієнтських замовлень
- **Фінанси** - облік доходів та витрат з балансом
- **Нотатки** - кольорові нотатки з тегами та зображеннями
- **Посилання** - менеджер закладок

---

## 💜 Про програму

Застосунок побудований як **односторінковий веб-застосунок** без використання фреймворків - виключно на чистому HTML, CSS та JavaScript.

Глобальний стан зберігається в пам'яті та синхронізується. Навігація між сторінками реалізована через динамічне показування/приховування елементів. Уся UI-логіка розбита на модульні JS-файли за зонами відповідальності.

Інтерфейс адаптований для мобільних пристроїв та підтримує **світлу і темну теми** з плавними CSS-переходами.

## ✨ Функціональні можливості

### 📋 Завдання
- Швидке додавання з рядка або через модальне вікно
- Пріоритети: Низький / Середній / Високий
- Дедлайни з підсвічуванням прострочених
- Підзавдання через кому
- Фільтри: Всі / Активні / Виконані / Важливі

### 🗂️ Kanban
- Три колонки: До виконання → В процесі → Виконано
- Переміщення карток між колонками кнопками ← →
- Теги для категоризації карток

### ⭐ Звички
- Тижневий трекер з відмітками днів Пн–Нд
- Стрік (кількість виконаних днів тижня)
- Прогрес-бар заповнення тижня

### 🎯 Цілі
- Категорії: Навчання / Кар'єра / Здоров'я / Фінанси / Особисте
- Range-слайдер для оновлення прогресу (0-100%)
- Дедлайн та анімований прогрес-бар

### ⏱️ Taймер (Pomodoro)
- Відлік до 25 хвилин з кільцем прогресу
- Старт / Пауза / Стоп
- Збереження сесій з назвою задачі та тривалістю

### 💰 Фінанси
- Записи типу «Дохід» або «Витрата»
- Категорії: Фриланс, Зарплата, Проект, Їжа, Транспорт...
- Підсумки: Загальний дохід / Витрати / Баланс

### 📝 Нотатки
- Кольорові картки з 6 варіантами кольору
- Теги, зображення (завантаження файлу)
- Пошук у реальному часі

### 🔗 Посилання
- Збереження URL з назвою та емодзі-іконкою
- Автододавання протоколу `https://`
